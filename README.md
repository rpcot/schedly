## Running the project

0. Clone the repository:  
```bash
git clone https://github.com/rpcot/raspisssanie.git
cd raspisssanie
```

1. Install yarn if you don't have it:
```bash
npm install --global yarn
```

2. Install dependencies:
```bash
yarn install
```

3. Start the bot:
```bash
yarn start
```

4. Start the API:
```bash
yarn api-start
```
